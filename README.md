# Flag-test

Vietnam flagg
Vietnam sitt flagg er 2:3. Det vil si at flagget er 30 enheter bredt og 20 enheter høyt.
Vi begynte med å lage en canvas hvor flagget skulle bli tegnet på, dette canvaset er 640px x 426px.
